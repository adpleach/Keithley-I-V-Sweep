# Keithley-I-V-Sweep
Python script to sweep the voltage on a Keithley SMU, and plot Current vs. Voltage.
Requires pyVISA: https://pyvisa.readthedocs.io/

Sweep parameters are at the top.  Option to save plots and data files.
